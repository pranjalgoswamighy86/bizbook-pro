# BizBook Pro — Desktop App Installation Guide

## System Requirements
- Windows 10 or Windows 11 (64-bit)
- Internet connection (the app connects to cloud database)
- Thermal printer or A4 printer connected

---

## Installation Steps

### Step 1: Download the Installer
1. Go to: https://github.com/pranjalgoswamighy86/bizbook-pro/releases/latest
2. Download `BizBook.Pro.Setup.2.1.2.exe` (or latest version)
3. Save it to your Downloads folder

### Step 2: Unblock the Installer (IMPORTANT — Windows Security)
Because the installer is downloaded from the internet, Windows blocks it by default.

1. Go to your **Downloads** folder
2. **Right-click** on `BizBook.Pro.Setup.2.1.2.exe`
3. Click **"Properties"**
4. At the bottom, check the **"Unblock"** checkbox
5. Click **"Apply"** then **"OK"**

### Step 3: Run the Installer
1. **Double-click** `BizBook.Pro.Setup.2.1.2.exe`
2. If Windows SmartScreen appears:
   - Click **"More info"**
   - Click **"Run anyway"**
3. Follow the setup wizard:
   - Choose installation location (default is fine)
   - Click **"Install"**
   - Wait for installation to complete
   - Click **"Finish"**

### Step 4: Launch BizBook Pro
1. Find **"BizBook Pro"** in your Start Menu
2. Click to launch
3. The app opens in a desktop window (not a browser tab)

### Step 5: Log In
1. Enter your BizBook Pro email and password
2. Click **"Sign In"**
3. You should see your Dashboard with all your data

## Setting Up Your Printer

### For Thermal Printers (58mm or 80mm)
1. Connect your thermal printer via USB
2. Install the printer driver (if not already installed)
3. Set it as your **default printer** in Windows Settings
4. In BizBook Pro, go to any Sale → click the **Print** button
5. The print preview will open showing the invoice
6. In the print dialog:
   - Select your thermal printer as the destination
   - Change paper size to **58mm** (or "Roll Paper")
   - Click **Print**

### For A4 Printers
1. Set your A4 printer as the default printer
2. In BizBook Pro, go to any Sale → click the **Print** button
3. The print preview will open
4. Click **Print** in the dialog

## Troubleshooting

### "Windows protected your PC" (SmartScreen)
1. Click **"More info"**
2. Click **"Run anyway"**
3. This is normal for new software without a code signing certificate

### App won't open / blank window
1. Close the app completely
2. Right-click the BizBook Pro shortcut → **"Run as administrator"**
3. If still blank, check your internet connection (the app needs internet)

### Print preview not showing
1. Make sure you're running the latest version (check Help → About)
2. The print preview opens in a new window — check your taskbar
3. If popups are blocked, allow popups for the app

### Printer not detected
1. Go to Windows Settings → Devices → Printers & scanners
2. Make sure your printer is listed and set as default
3. Try printing a test page from Windows first

## Updating to a New Version

1. Download the new `.exe` from the releases page
2. **Unblock** the file (Step 2 above)
3. Run the installer — it will update the existing installation
4. Your data is safe (stored in the cloud, not on your PC)
